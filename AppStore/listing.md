## Description
PageReset makes hostile webpages usable again.

Safari is fast and private — until a site disables selection, kills right-click, or drops three overlays on your article. PageReset restores basic browsing hygiene with a compact toolkit:

• Restore text selection  
• Restore right-click  
• Remove distracting overlays (and zap stubborn elements)  
• Copy without formatting  
• Copy selection or page as Markdown  
• Extract tables as CSV  
• Copy all links  
• Per-site rules and keyboard shortcuts  

**One-time purchase. No account. No subscription. All processing local.**

PageReset does **not** bypass paywalls, login walls, DRM, or paid content. It focuses on everyday web cleanup: reading, selecting, and copying.

### How to enable
1. Open PageReset once after install.  
2. Click “Open Safari Settings…” and enable the PageReset extension.  
3. Grant access for the sites you want to clean up.  
4. Use the toolbar button for actions and per-site toggles.

### Shortcuts (customizable in Safari Settings → Extensions → PageReset)
• ⌘⇧C — Copy without formatting  
• ⌘⇧M — Copy as Markdown  
• ⌘⇧X — Remove overlays
